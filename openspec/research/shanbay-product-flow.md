# 扇贝单词产品流程调研

> 调研时间：2026-04-26，来源：搜索结果综合整理

## 学习流程

1. 首页展示今日计划（新词数量），点击「开始学习」进入
2. 每张卡片先只显示单词，用户判断**认识 / 不认识**：
  - **认识**：直接进入单词详情页，标记为已掌握，不再复习
  - **不认识**：依次展示例句 → 英文释义 → 中文释义，将该词加入本日列表末尾稍后再学
3. 每 **7 个单词**为一组，组结束后显示总结（没记住的标红）
4. 全部单词完成后打卡，回首页

## 复习流程

- **旧词复习**：7 天前学过的旧词，模式为「看中文辨英文」，一组 7 个
- **极速刷词**：闪卡模式，看词选义，快速多轮刷
- **拼写测试**：看义拼写，检验拼写掌握度
- 没有旧词时从当前词书随机出词

## 生词本

- 用户点「不认识」的词自动进入生词本（无需手动添加）
- 生词本作为复习词池的主要来源

## 核心设计理念

- 用户**自主判断**认识/不认识，保证效率
- **学过才能复习**：复习词池来自已学单词，而非整个词库
- 7词一组 + 打卡 = 小目标激励
- 社交元素（小组、同桌）促进长期使用

## 与拾词的对比


| 维度    | 扇贝            | 拾词（当前）      |
| ----- | ------------- | ----------- |
| 学习单元  | 每日计划（N词）      | 按章节浏览       |
| 学习判断  | 认识/不认识        | 纯浏览，无判断     |
| 生词本来源 | 学习时点「不认识」自动加入 | 复习时点「不认识」加入 |
| 复习来源  | 已学单词（生词本）     | 章节 or 生词本   |
| 记忆算法  | AI + 艾宾浩斯曲线   | 无           |
